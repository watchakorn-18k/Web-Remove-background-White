# Setup
```
git clone https://github.com/watchakorn-18k/Web-Remove-background-White.git
cd Web-Remove-background-White
```



# Install Package 
## UNIX
```
virtualenv env
source env/Scripts/activate
pip install -r requirements.txt
```
## WINDOWS
```
virtualenv env
.\env\Scripts\activate
pip install -r requirements.txt
```

# Start Server
```
python app.py
```
